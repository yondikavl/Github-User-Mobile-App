## Features:

- Viewing a list of GitHub users
- Searching for users
- Viewing user details
- Adding users to favorites
- Changing themes
  
- Using MVVM (Model-View-ViewModel) Architecture
- Using ViewBinding
- Using Retrofit to request API network
